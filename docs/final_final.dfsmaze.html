Заимствовано. Создает лабиринт, начиная с заданной точки.

Parameters
maze:list[list]Список списков, представляющий лабиринт. Каждая ячейка может быть пустой (" "), стеной или другой преградой.
height:intВысота лабиринта.
width:intШирина лабиринта.
start_x:intНачальная строка для начала поиска.
start_y:intНачальный столбец для начала поиска.
Returns
None. Функция изменяет лабиринт in-place, удаляя стены и открывая проходы.
Raises
IndexErrorЕсли начальная точка выходит за пределы лабиринта.
ValueErrorЕсли начальная точка не является пустой клеткой (" ").
